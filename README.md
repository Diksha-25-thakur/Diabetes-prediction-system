# 🩺 Diabetes Prediction System

## 📌 Project Overview

This project is a Machine Learning-based web application that predicts whether a person is diabetic or not using health parameters such as glucose level, BMI, blood pressure, and age. It helps in early detection and supports quick decision-making.

---

## 🎯 Objective

* Predict diabetes using Machine Learning
* Provide early detection support
* Create a user-friendly web interface
* Display results with confidence level

---

## 🧠 Machine Learning Model

* Algorithm: Logistic Regression
* Dataset: Pima Indians Diabetes Dataset
* Accuracy: ~78%

---

## ⚙️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Database: SQLite
* Libraries: NumPy, Pandas, Scikit-learn

---

## 🚀 Features

* User-friendly interface
* Instant prediction (Diabetic / Not Diabetic)
* Confidence percentage display
* Prediction history storage
* Data visualization using charts

---

## 📊 How It Works

1. User enters health details
2. Data is sent to the Flask backend
3. Model processes input using Logistic Regression
4. Prediction result is generated
5. Result is displayed with confidence

---

## 📁 Project Structure

* app.py → Main backend file
* model.pkl → Trained ML model
* templates/ → HTML files
* static/ → CSS, JS, images
* diabetes.csv → Dataset

---

## ▶️ How to Run the Project

1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000/
